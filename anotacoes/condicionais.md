#Condicionais: if / else

As condicionais if e else são usadas para fazer o código tomar decisões.

if significa “SE” uma condição for verdadeira, execute determinado bloco de código.

else significa “SE NÃO”, execute outro bloco de código quando a condição não for atendida.

É assim que o programa se comporta de forma diferente dependendo da situação.
Exemplo: se o usuário digitar algo, faço uma coisa; se não digitar, faço outra.

As condicionais permitem controlar o fluxo do código e criar interações mais inteligentes dentro da aplicação.
