# DOM – Anotações

## O que é DOM?
É a representação do HTML em forma de árvore, permitindo manipular elementos via JavaScript.

## Selecionando elementos
- `document.getElementById()`
- `document.querySelector()`

## Alterando conteúdo
- `element.textContent`
- `element.innerHTML`

## Eventos mais comuns
- `onclick`
- `addEventListener('click', callback)`
