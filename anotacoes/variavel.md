# Variável em JavaScript

O que é uma variável? 

É um espaço na memória onde guardamos um valor para usar no nosso código depois.

EX: let nome = "João";

let → palavra usada para declarar uma variável.

nome → identificador, o nome que damos para acessar esse valor.

= → sinal de atribuição (coloca o valor dentro da variável).

"João" → o valor armazenado (uma string).

Como usar: 

document.getElementById("mensagem").textContent = "Olá " + nome;

Olá João

# Observações importantes

O valor pode ser alterado depois se você usar let.

Variáveis ajudam a evitar repetição desnecessária.

Usar nomes claros deixa o código mais fácil de entender.
